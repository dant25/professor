Programação C++

C++ é uma linguagem de programação compilada multi-paradigma (imperativa e orientada a objetos). A linguagem é considerada de baixo/médio nível, pois combina características de linguagens de alto e baixo níveis.

Banco de Dados ( BD )

Bancos de Dados são um conjunto de dados relacionados entre si que, em geral, são acessados atravez de um sistema de gerenciamento de banco de dados (SGBD), que possui recursos capazes de manipular as informações do BD e interagir com o usuário.
