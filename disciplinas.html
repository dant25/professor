Disciplinas

Uma descrição rápida.

Algoritmos Computacionais: tem como objetivo dominar os fundamentos do raciocínio algorítmico e determinístico para a resolução de problemas utilizando o computador. O aluno irá conhecer as principais estruturas de dados com ênfase em seus algoritmos. Ao final o aluno estará apto a desenvolver aplicações utilizando uma linguagem de programação.

Arquitetura de Computadores: tem como objetivo capacitar os alunos a compreenderem a organização e arquitetura básica de um computador, em especial, capacitar os alunos a entenderem e analisarem criticamente as opções para definição do conjunto de instruções de uma arquitetura, da estrutura da CPU e dos componentes de memória e barramentos.

Banco de Dados: tem como objetivo utilizar as principais técnicas de modelagem para projetar, criar, analisar e otimizar Banco de Dados relacionais usando sistemas de gerenciamento de banco de dados de última geração.

Matemática Discreta: tem como objetivo desenvolver nos alunos a capacidade de raciocinar com o abstrato.

Programação de Computadores: tem como objetivo aplicar conhecimentos de algoritmos no desenvolvimento de programas de computadores, Utilizando conceitos básicos de orientação a objetos.
